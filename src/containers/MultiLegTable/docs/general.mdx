# MultiLegTable

`data-display`

支持多腿查看和编辑的表格

## 使用场景

展示多腿结构数据下

当需要对多腿结构进行联动，计算，操作等行为时候

## 功能交互设计

- 单元格状态分为：可编辑、不可编辑（置灰）、无此字段
- 表头样式、横向滚动时表头固定
- 固定腿宽，超长内容elip悬停展示全部内容
- tag样式：标的物列表之外下拉选项上屏后使用tag样式展示
- 切换是否年化时使用switch组件，或直接改成下拉选项
